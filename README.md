# 100 Days of Python 

Documenting my Python journey through **Angela Yu's 100 Days of Code** bootcamp - 
one project at a time, built while working full-time and transitioning into AI development.

## Why this exists

I'm a career changer with a background in operations, team leadership, and front-end 
development. Python is the bridge between where I've been and where I'm going - 
AI-focused software development. This repository is my public record of 
showing up, learning, and building.

Not every day will have a commit. But every project here was built by hand, understood, 
and reflected on.

## What I'm learning

| Area | Topics |
|------|---------|
| Foundations | Variables, data types, strings, input/output |
| Logic | Conditionals, loops, functions, scope |
| Data structures | Lists, dictionaries, tuples, sets |
| OOP | Classes, inheritance, instances |
| Intermediate | File I/O, error handling, APIs, web scraping |
| Projects | 100 progressively complex Python projects |
